La palabra de configuracion (config bits o fuses) define el comportamiento del hardware al encender: oscilador, watchdog, MCLR, brown-out, etc.

<AIExplain title="En MPASM">
Usas directivas `CONFIG` al inicio del programa. El programador graba estos bits en la Flash de configuracion, no en el codigo ejecutable.
</AIExplain>

## Ejemplo tipico (cristal 20 MHz)

```asm
        CONFIG  FOSC = HS        ; Oscilador de alta velocidad
        CONFIG  WDT  = OFF       ; Watchdog deshabilitado
        CONFIG  LVP  = OFF       ; Low Voltage Programming off
        CONFIG  MCLRE = ON       ; Pin MCLR habilitado
        CONFIG  PBADEN = OFF     ; Puertos digitales al inicio
        CONFIG  BOR  = OFF
        CONFIG  PWRT = ON
```

## Sistema minimo

Segun el Tema 2 del el curso de Microcontroladores (UNEXPO), el sistema minimo incluye:

- PIC18F4550
- Cristal de 20 MHz + condensadores (tipicamente 1 nF cada uno)
- Reset con pull-up de 10 kΩ en MCLR
- Alimentacion 5 V (VDD/VSS)
- Condensador de desacople 10 nF cerca del PIC
