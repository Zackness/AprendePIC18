## Single loop

```asm
RETARDO
    MOVLW   .N
    MOVWF   CONTADOR
BUCLE
    DECFSZ  CONTADOR,F
    BRA     BUCLE
    RETURN
```

With `CALL`: **Σ CI ≈ 3N + 4**. Body only: **≈ 3N + 2**.

### PDF example: N=200 @ 20 MHz

Σ CI = 605 → delay = **121 µs**.

<AIExplain title="Common mistake">
Do not count `DECFSZ` as N×1 only. The final iteration when counter hits 0 costs **2 cycles**.
</AIExplain>

## Two nested loops (PDF structure)

<Figure
	src="/images/pdf/tema-4-5/page-8.png"

/>

```asm
    MOVLW   .M
    MOVWF   CONTADOR_2
BUCLE_2
    MOVLW   .N
    MOVWF   CONTADOR_1
BUCLE_1
    DECFSZ  CONTADOR_1,F
    BRA     BUCLE_1
    DECFSZ  CONTADOR_2,F
    BRA     BUCLE_2
    RETURN
```

**Σ CI ≈ 3NM + 4M + 5** (no CALL).

## Three nested loops

**Σ CI ≈ 3NMP + 4MP + 4P + 5**
