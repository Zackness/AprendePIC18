## How to explain it in an exam

A **microcontroller** integrates CPU, memory, and peripherals in one chip to control devices. A **microprocessor** usually needs external memory and peripherals, and can run full operating systems.

The PIC18F4550 is academically useful because it shows how hardware is configured through registers without relying on high-level libraries.
