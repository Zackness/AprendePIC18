## Key concepts

- Timer as external pulse counter
- Display multiplexing to save pins
- Persistence of vision with fast refresh (~50 Hz per digit)
- Formula: frequency = count / window time

## Course links

Combines Timer0/Timer1 counter mode, GPIO and possibly interrupts for refresh.
