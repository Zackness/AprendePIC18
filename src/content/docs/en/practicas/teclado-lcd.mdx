## Connections

| Device | PIC pins |
| --- | --- |
| LCD data (4 bit) | RD4-RD7 |
| LCD RS, E | RD2, RD3 |
| Keyboard rows | RB0-RB3 |
| Keyboard columns | RB4-RB7 |

## Program requirements

- MPASM assembler, modular and commented
- RB4-RB7 interrupts for keyboard
- LCD.INC library (4-bit bus) per Topic 7.1
- Pseudocode and flowchart in report
