This practice explains the basic idea of digital output: a pin must be configured as an output and then receive a logic value.

<PracticeBox title="Turn on an LED on RB0" level="Beginner">
Goal: configure `RB0` as output and write a high level to turn on an LED connected correctly with its resistor.
</PracticeBox>

<AIExplain title="Concept before code">
`TRISB` decides whether each PORTB pin is input or output. A bit set to `1` means input; a bit set to `0` means output. To use `RB0` as output, bit 0 of `TRISB` must be `0`.
</AIExplain>

## Base code

```asm
        LIST    P=18F4550
        #include <P18F4550.INC>

        CONFIG  FOSC = HS
        CONFIG  WDT = OFF
        CONFIG  LVP = OFF

        ORG     0x0000
        GOTO    start

start:
        CLRF    LATB        ; Clear PORTB outputs
        BCF     TRISB, 0    ; RB0 as output
        BSF     LATB, 0     ; RB0 high

loop:
        GOTO    loop

        END
```

## What you should be able to explain

1. Why `TRISB` is used.
2. Why `BCF TRISB, 0` makes `RB0` an output.
3. Why the code writes to `LATB` and not necessarily to `PORTB`.
4. Why the program ends in an infinite loop.

## Exercise

Modify the program so the LED is on `RB1`. Then explain which lines changed and why.
