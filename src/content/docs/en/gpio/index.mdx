`TRISx` sets direction. `LATx` writes outputs. `PORTx` reads pins.

```asm
        BCF     TRISB, 0
        BSF     LATB, 0
```

Lab Practice 1 uses ports B/D as inputs and C/E as LED outputs with RA0 as operation selector.
