## Greater than (A &gt; B)

```asm
MOVF    B,W
SUBWF   A,W
BTFSS   STATUS,C
GOTO    A_NO_ES_MAYOR_B
BTFSS   STATUS,Z
GOTO    A_ES_MAYOR_B
```

## Summary table A − B

| Case | C | Z |
| --- | --- | --- |
| A = B | 1 | 1 |
| A &gt; B | 1 | 0 |
| A &lt; B | 0 | 0 |

## REG1 &gt; REG2 subroutine

```asm
    MOVF    REG2,W
    SUBWF   REG1,W
    BTFSS   STATUS,C
    GOTO    FIN_COMP
    BTFSS   STATUS,Z
    GOTO    MAYOR
FIN_COMP
    RETURN
MAYOR
    RETURN
```
