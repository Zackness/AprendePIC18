This course focuses mainly on **MPLAB v8** because it is often more compatible with classic MPASM and academic project workflows. **MPLAB X** will also be explained so you understand the modern environment.

<AIExplain title="Real workflow">
You write assembly in MPLAB, compile a `.hex` file, load it into Proteus for simulation, and finally burn it into a physical board if you have the hardware.
</AIExplain>

<AIExplain title="Next step on the path">
After this page, the [study path](/en/ruta-de-estudio/) takes you to the ports practical guide and the [First LED tutorial](/en/tutoriales/primer-led/), where you create the MPLAB project and simulate in Proteus step by step (setup, CONFIG, TRIS, and LAT).
</AIExplain>

<Tabs>
	<TabItem label="MPLAB v8">
		1. Create a project with the wizard.
		2. Select `PIC18F4550`.
		3. Use MPASM as the assembler tool.
		4. Add the `.asm` file.
		5. Build and generate the `.hex` file.
	</TabItem>
	<TabItem label="MPLAB X">
		1. Create a standalone project.
		2. Select the device family and `PIC18F4550`.
		3. Configure a compatible toolchain.
		4. Add assembly code.
		5. Build and review errors in the output panel.
	</TabItem>
	<TabItem label="Proteus">
		1. Place the PIC18F4550.
		2. Connect crystal, power, and components.
		3. Load the `.hex` file in the PIC properties.
		4. Set the clock frequency.
		5. Run the simulation and observe the pins.
	</TabItem>
	<TabItem label="Physical board">
		1. Check power supply.
		2. Verify oscillator, MCLR, and connections.
		3. Program the microcontroller.
		4. Measure outputs with a multimeter or LED.
		5. Compare real behavior against Proteus.
	</TabItem>
</Tabs>
