## Supporting theory

Before coding, review [PWM / CCP](/en/pwm/): **PR2** calculation, **duty cycle**, and **Timer2** setup.

## What your code should do

1. Calculate `PR2` for the target PWM frequency (e.g. ~1 kHz).
2. Configure `T2CON`, `PR2`, and `CCP1CON` in PWM mode.
3. Write duty to `CCPR1L` and `CCP1CON` bits.
4. Connect load on **RC2/CCP1** and vary duty (potentiometer + ADC optional).
