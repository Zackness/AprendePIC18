<AIExplain title="Digital voltage">
With Vref=5V and 10 bits, 1 LSB ≈ 4.88 mV. Result in ADRESH:ADRESL per ADFM format.
</AIExplain>

## Analog channels

| Port | Pins |
| --- | --- |
| A | RA0/AN0, RA1/AN1, RA2/AN2, RA3/AN3, RA5/AN4 |
| B | RB0/AN12, RB1/AN10, RB2/AN8, RB3/AN9, RB4/AN11 |
| E | RE0/AN5, RE1/AN6, RE2/AN7 |

**Note:** RA4 is not analog.

## Registers

- `ADCON0` — channel, GO/DONE, ADON
- `ADCON1` — digital/analog configuration (PCFG)
- `ADCON2` — conversion clock and format (ADFM)
- `ADRESH`, `ADRESL` — result

## Conversion steps

1. Configure analog inputs (`ADCON1`)
2. Set pins as inputs (`TRIS`)
3. Select channel and enable ADC (`ADCON0`)
4. Wait acquisition time (TACQ)
5. Start conversion (`GO/DONE = 1`)
6. Wait for `GO/DONE = 0`
7. Read `ADRESH:ADRESL`
