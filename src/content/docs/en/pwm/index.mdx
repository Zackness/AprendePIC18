The CCP module has three modes: **Capture**, **Compare**, and **PWM**. PWM is the most common for motor speed and LED dimming.

<AIExplain title="CCP pins">
CCP1 on RC2, CCP2 on RC1. PWM uses Timer2 as the time base.
</AIExplain>

## Formulas (Topic 9 CCP)

**Period:**
`T_PWM = 4 * (PR2 + 1) * Tosc * Timer2 prescaler`

**Duty cycle:**
`CCPR1L:CCP1CON<5:4> = (T_PWM * X%) / (Tosc * prescaler)`

## PWM setup (5 steps)

1. `BCF TRISC, 2` — RC2 as output
2. Load PR2 for target frequency
3. Configure T2CON (prescaler, TMR2ON)
4. Load CCPR1L and DC1B1:DC1B0 bits
5. `CCP1CON = 0x0C` — PWM mode (CCP1M3:CCP1M0 = 11xx)
