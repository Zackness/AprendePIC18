## Learn like a course, search like documentation

AprendePIC18 is a web platform built by **zackness**, maintained by **[StartupVen](https://startupven.com)** and part of the **[CALETAS](https://caleta.top)** ecosystem, that organizes the **Microcontrollers** course material from **Mechatronics Engineering** at **UNEXPO** to study the **PIC18F4550**: assembly, registers, MPLAB, Proteus, physical boards, and exam preparation.

<AIExplain title="Project goal">
The goal is not to replace the datasheet. The goal is to translate it into a clear learning path: first understand the concept, then inspect the register, then build a practice, and finally solve exam-style questions.
</AIExplain>

## Quick map

1. Start with [Study path](/en/ruta-de-estudio/) — theory and tutorials in order (Getting started → Foundations → course).
2. Foundations: [architecture](/en/fundamentos/arquitectura/), [registers](/en/fundamentos/registros/), [assembly](/en/fundamentos/ensamblador/), [MPLAB & Proteus](/en/fundamentos/mplab-proteus/), then [First LED](/en/tutoriales/primer-led/) on the path.
3. Peripherals: [GPIO](/en/gpio/), [timers](/en/timers/timer0/), [interrupts](/en/interrupciones/), [ADC](/en/adc/), [PWM](/en/pwm/), [serial](/en/comunicacion/uart/).
4. Catalog: [all tutorials](/en/tutoriales/) when you need a specific lab.
5. Exam: [theory guide](/en/parcial/guia-teorica/), [exercises](/en/parcial/ejercicios/), [glossary](/en/referencia/glosario/).
