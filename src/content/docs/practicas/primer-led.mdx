Esta practica busca entender la idea basica de salida digital: un pin debe configurarse como salida y luego recibir un valor logico.

<PracticeBox title="Encender un LED en RB0" level="Inicial">
Objetivo: configurar `RB0` como salida y escribir un nivel alto para encender un LED conectado correctamente con su resistencia.
</PracticeBox>

<AIExplain title="Concepto antes del codigo">
`TRISB` decide si cada pin del puerto B es entrada o salida. Un bit en `1` significa entrada; un bit en `0` significa salida. Para usar `RB0` como salida, el bit 0 de `TRISB` debe estar en `0`.
</AIExplain>

## Codigo base

```asm
        LIST    P=18F4550
        #include <P18F4550.INC>

        CONFIG  FOSC = HS
        CONFIG  WDT = OFF
        CONFIG  LVP = OFF

        ORG     0x0000
        GOTO    inicio

inicio:
        CLRF    LATB        ; Limpia salidas del puerto B
        BCF     TRISB, 0    ; RB0 como salida
        BSF     LATB, 0     ; RB0 en alto

bucle:
        GOTO    bucle

        END
```

## Que debes poder explicar

1. Por que se usa `TRISB`.
2. Por que `BCF TRISB, 0` convierte `RB0` en salida.
3. Por que se escribe en `LATB` y no necesariamente en `PORTB`.
4. Por que el programa termina en un bucle infinito.

## Ejercicio

Modifica el programa para que el LED este en `RB1`. Luego explica que lineas cambiaron y por que.
