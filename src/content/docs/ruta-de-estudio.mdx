## Regla de oro

No memorices tablas completas al principio. La habilidad clave del parcial es responder:

**Que quiero controlar → que registro lo configura → que bit debo modificar**

## Material de referencia del curso

El contenido de AprendePIC18 se alinea con estos materiales del curso UNEXPO:

| Tipo | Archivos |
| --- | --- |
| Teoria | Tema 1, Tema 2-3, Tema 6-10, instrucciones PIC |
| Laboratorio | Practica 1, 2, 3, 6, 7, 8 |
| Parcial | Ejercicios II parcial 20252 |
| Referencia | PIC18LF4455-I-PT.PDF (datasheet) |

AprendePIC18 traduce ese material a paginas navegables, diagramas y tutoriales interactivos.
